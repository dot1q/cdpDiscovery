About

This repo is simply an example for some angular testing. A couple of goals of mine were:

  • Angular ES6
  • UI Router
  • Vis JS Angular

In addition, gulp and build tasks were implemented to make life easier :)

  • Browser sync on edits
  • Webpack, dawg

Purpose

This is a proof of concept to see if on the fly network mapping could solve the problem of an ever changing network. Yes, documentation is great, but how much greater would it be if we could spend less time documenting and more time drinking beer?
